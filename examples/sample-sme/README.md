# サンプル: 株式会社さくら製作所（架空）

架空の中小製造業3期分（FY2021〜FY2023）の決算書サンプル。単位は千円。

## シナリオ

主力製品の価格競争激化と原材料高により、3期連続で業績が悪化していくシナリオ。

- 売上高: 500,000 → 480,000 → 430,000（3期で▲14%）
- 売上高総利益率: 30.0% → 29.17% → 25.58%（原価率の悪化＝価格転嫁できていない）
- 売上高営業利益率: 10.0% → 8.75% → 3.49%（販管費も高止まりし収益性が急速に悪化）
- 自己資本比率: 50.0% → 48.99% → 43.75%（利益の目減りで内部留保の積み上げが止まり後退）
- 現金及び預金: 60,000 → 50,000 → 30,000（資金繰りの悪化が数字に表れ始めている）

診断士的な着眼点としては「増収減益ではなく減収減益」「原価率と販管費率の両方が悪化」「当座資産（特に現金）の減少ペースが売上減少より速い」という複合的な悪化パターンになっている。
