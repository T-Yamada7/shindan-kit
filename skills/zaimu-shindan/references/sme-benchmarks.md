# 中小企業ベンチマーク値

## 数値の唯一の情報源

**ベンチマーク数値そのものの唯一の情報源（single source of truth）は `sme-benchmarks.json` である。**
`scripts/benchmark.py` は実行時にそのJSONを読み込む。以前は `benchmark.py` 内に同じ値のPython辞書を
ハードコードしており、コードとドキュメントで値が食い違う（二重管理）リスクがあったため、JSON側だけを
編集すれば良い構成に変更した。**このファイルの表は人間が読むための複製であり、値を変更する際は
必ず `sme-benchmarks.json` を先に更新し、この表をそれに合わせること。**

## 出典についての重要な注意（MVP時点の暫定状態）

**`sme-benchmarks.json` の数値は Phase 1 (MVP) 用の暫定サンプル値であり、公的統計からの正式な引用ではない。**
中小企業庁「中小企業実態基本調査」等の公的統計の実数値をまだ転記していない状態でリポジトリを進めている。
Phase 1完了までに、または遅くともPhase 2着手前までに、以下を行うこと:

1. 中小企業庁「中小企業実態基本調査」（最新年度、業種別）から該当指標の実数値を取得する
2. 取得した値で `sme-benchmarks.json` の `benchmarks` を置き換え、`source_status` に
   **調査年度・業種区分・URLを出典として明記**する
3. このファイルの表を `sme-benchmarks.json` の値に合わせて更新する
4. 有料データベース・非公開データの値は**絶対に転載しない**（README/リスク対処方針に準拠）

暫定値は「全産業・全規模の中小企業のおおよその中央値イメージ」を診断士実務の一般的な相場観から仮置きしたものであり、業種別の精度は保証しない。デモ・開発用途に限定して使うこと。

## 業種別ベンチマークについて（Phase 2以降）

v0.1は全業種一律の単一ベンチマークで動く。業種別ベンチマーク（製造業/卸売業/小売業/サービス業等での比較）はPhase 2以降で対応する。業種を指定できるようにする場合も、出典の調査における業種区分の定義に厳密に従うこと（自己流の業種区分に当てはめない）。

## 所見執筆時の注意

- ベンチマークとの差分はあくまで「相対評価の参考」であり、絶対的な合否判定ではない
- `benchmark.py` の出力には `benchmark_source` フィールドに出典の状態（暫定/正式）が入るので、レポート中でも「暫定値のため参考情報」である旨を明示すること
